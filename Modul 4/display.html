Ini adalah elemen pertama menggunakan DIV
Ini DIV di dalam DIV namun membuat elemen blok baru Ini SPAN di dalam DIV tidak membuat elemen blok baru

HTML dan CSS tersebut menunjukkan penggunaan elemen div dan span serta pengaturan tampilannya dengan CSS. Elemen div digunakan sebagai elemen blok yang membuat baris baru, sedangkan span biasanya elemen inline, tetapi pada kode ini diubah menjadi blok menggunakan display: block sehingga tampil seperti elemen blok. Selain itu, CSS juga digunakan untuk mengatur ukuran, margin, border, dan warna background agar tampilan halaman lebih jelas.
