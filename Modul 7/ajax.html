Uraian AJAX & jQuery

1. Apa itu AJAX?

AJAX (Asynchronous JavaScript and XML) adalah teknik pengembangan web yang memungkinkan halaman memperbarui data secara asynchronous, tanpa perlu reload halaman.

2. Apa itu jQuery?

jQuery adalah library JavaScript yang mempermudah manipulasi HTML, event, animasi, dan AJAX.

3. Cara Menggunakan jQuery

jQuery dapat digunakan dengan menambahkan library melalui CDN seperti:
<script src="https://code.jquery.com/jquery-3.6.0.min.js"></script>
